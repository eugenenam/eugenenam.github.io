# eugenenam.github.io

Personal portfolio and website for [Eugene Nam](https://eugenenam.com). Hosted on GitHub Pages.

## Stack

Static HTML, CSS, and JavaScript. No build step, no framework, no dependencies to install.

## Run locally

```
node serve.js
```

Opens at http://localhost:8081.

## Structure

| Path | Purpose |
|------|---------|
| `index.html` | Main page (single-page layout) |
| `css/style.css` | All custom styles |
| `js/main.js` | All scripts |
| `portfolio/` | Individual case study pages |
| `img/` | Images and assets |
| `medium-posts.json` | Medium article feed data |

## Deploy

Push to `master`. GitHub Pages auto-deploys to [eugenenam.com](https://eugenenam.com).
